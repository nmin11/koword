## Introduction

영단어 공부를 위한 CLI 프로그램입니다.

## Commands

### `koword guess`

- 주어진 한글 뜻에 맞는 영단어 맞추기 퀴즈
- `--count` `-c` : 문제 개수 지정 (기본값: 20)

### `koword mean`

- 주어진 영단어에 맞는 한글 뜻 맞추기 퀴즈
- `--count` `-c` : 문제 개수 지정 (기본값: 20)

### `koword wordle`

- 영단어 추측하기 게임
- `--length` `-l` : 단어 길이 지정
- `--tries` `-t` : 시도 횟수 지정 (기본값: 6)
